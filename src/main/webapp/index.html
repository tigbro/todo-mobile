Todo-App

This site provides a simple todo application in the following technologies:
  • Plain jQuery Mobile without databinding: Here
  • Desktop version with angularJS databinding: Here
  • jQuery Mobile version with angularJS databinding: Here
  • Plain Sencha Touch version: Here
  • Sencha Touch version with angularJS databinding: Here
All angular versions share a common javascript controller. For the backend, we use the Open Key Value Store, see http://openkeyval.org/

For the adapter between anglar and jquery mobile see here: http://github.com/tigbro/jquery-mobile-angular-adapter

For the adapter between anglar and sencha touch see here: http://github.com/tigbro/sencha-touch-angular-adapter
